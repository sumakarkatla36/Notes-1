git is a content tracker

Ex: man git 

git hash-object --stdin : it will give commit id

example: $ echo "Hello world" | git hash-object --stdin
commitid or SHA Code : 802992c4220de19a90767f3000a79a31b98d0df7

whatever we store in git we will get a SHA code.

git is a key value pair.

commit ID is the key
content is the value

if we change the content the commit id will also change

echo "content" | git hash-object -w --stdin

git cat-file commit-id -p --> you will get total information
